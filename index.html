邮件工作流

支持通用邮箱配置、IMAP/POP3/SMTP、刷新收信、发送邮件、草稿同步、存档、垃圾邮件、星标和已读状态。

AI 问答

AI 回答基于已经向量化的邮件数据，支持 Markdown 排版和相关邮件引用，点击即可回到原始邮件。

本地向量化

向量化只使用 Apple NaturalLanguage NLEmbedding，不使用远端 embedding 模型，也不上传邮件内容生成向量。

附件体验

支持附件名称解码、系统文件图标、打开附件、另存为，以及可读取附件内容参与本地索引。

Privacy First

Account passwords, OAuth tokens, and API keys are stored in macOS Keychain. AI chat is optional; regular mail features work without configuring an API key.
